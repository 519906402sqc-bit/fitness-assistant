# 薄肌训练助手

一个为 175cm / 63kg / 每周 4 练 + hiphop 辅助训练定制的训练小助手。

## 功能

- 输入 `训练日2`、`今天背部练什么` 等指令，查看当天训练
- 查看每个动作的标准步骤、关键口令、常见错误和教学链接
- 记录每组重量、次数、休息时间和 RPE，并反馈目标肌肉感觉、疼痛与跳舞影响
- 根据最近真实记录和恢复反馈生成下次训练建议
- 支持 iPhone Safari 添加到主屏幕，作为 PWA 使用

## GitHub Pages 发布

1. 在 GitHub 创建一个公开仓库，例如 `fitness-assistant`
2. 上传本文件夹中的所有文件
3. 打开仓库 `Settings` -> `Pages`
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，文件夹选择 `/root`
6. 保存后等待 1-3 分钟

发布后访问：

`https://你的GitHub用户名.github.io/fitness-assistant/`

在 iPhone Safari 打开后，可以通过分享按钮选择“添加到主屏幕”。
